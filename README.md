# Junior-2526-Monte-Carlo
Репозиторий проекта "Создание программы для определения распространения лазерного излучения в неоднородных тканях"

На основе физической модели и метода Монте-Карло сформирована математическая модель переноса фотонов в тканях. Для каждого фотона учитываются свободные пробеги между взаимодействиями, поглощение, рассеяние и выход за пределы моделируемой области. Статистическое усреднение большого числа траекторий определяет итоговую физику процесса.

Модель поддерживает неоднородную среду (слои дермы, эпидермиса и гиподермы, а также опухоли). Результаты представлены в виде графиков: распределение конечных координат фотонов в поперечном срезе и зависимость плотности энергии от глубины.


MC_main.py
  - Назначение: точка входа в программу. Читает параметры конфигурации, инициализирует модель и запускает симуляцию, собирает результаты.
  - Вход: клавиатура и тачпад компьютера.
  - Выход: результаты моделирования (графики).

MC_algo.py
  - Назначение: реализация алгоритма Монте-Карло переноса фотонов. Содержит цикл моделирования траекторий фотонов.
  - Вход: параметры моделирования (коэффициенты среды, число фотонов, геометрия).
  - Выход: энергетическая плотность по глубине, массив конечных координат фотонов.
